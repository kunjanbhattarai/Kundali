This is a Flask-based Kundali generation system that computes planetary positions, Navamsa (Naba Kundali), and allows AI-powered astrology chat.

## Features
- Calculates **Janma Kundali** based on birth details.
- Computes **Navamsa (D9)** divisional chart.
- AI chatbot provides astrology insights in **Nepali and English**.
- Simple **Flask API** for web and mobile access.
